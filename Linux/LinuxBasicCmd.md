# Linux基本命令

1. 显示当前工作路径
(Print Working Directory)
```linux
$ pwd
```
2. 列出当前工作路径下所有可见文件夹
(list)
```linux
$ ls
```
3. 切换至不同路径
(Change Directory)
```linux
$ cd *想切换到的路径*
```
4. 复制文件（至某处）
(copy)
```linux
$ cp *想复制某(绝对/相对)路径下的文件* *(想复制到的路径,省略则为当前路径)下的(输出文件名省略则为原来的文件名)*
```
5. 删除文件
(remove)
```linux
$ rm *想删除(路径下)的文件(夹)1* *想删除(路径下)的文件(夹)2* *3* *4 ...*
```

> 按Tab键可以自动补全
